# NEONSKI ODBOJ ⚡

## Putanja svetlosti

Neonski Odboj je arkadna browser igra u kojoj igrač odbija lopticu, uništava mete, skuplja poene i bori se protiv moćnih boss protivnika.

## 🎮 O igri

Igra sadrži:

- ⚡ Neon grafiku i efekte
- 🟢 Fiziku odbijanja lopte
- 🎯 Uništavanje meta
- 💥 Efekte čestica
- 🕳️ Crne rupe kao prepreke
- 👾 Boss protivnike
- 🛡️ Štit i posebne sposobnosti
- 🔊 Zvučne efekte
- 🏆 Sistem poena i combo bonusa
- ❤️ Sistem života
- 🌊 Talase igre

## 👾 Boss protivnici

U igri postoje različiti tipovi bossova:

- 🟣 Ljubičasti boss
- 🔴 Crveni boss
- 🔵 Plavi boss sa štitom
- 🟢 Zeleni boss
- 🟡 Zlatni boss

Svaki boss ima različite karakteristike, brzinu, energiju i napade.

## 📱 Podrška za uređaje

Igra radi na:

- 🖥️ Računarima
- 💻 Laptopovima
- 📱 Mobilnim telefonima
- 📲 Tabletima

Podržane kontrole:

- 🖱️ Miš
- 👆 Dodirni ekran

Interfejs se prilagođava različitim veličinama ekrana.

## 🎯 Cilj igre

- Uništiti mete
- Sakupljati poene
- Povećavati combo
- Izbegavati crne rupe
- Poboljšavati rekord
- Pobedititi bossove
- Stići do završnog talasa

Igra se pokreće u internet pregledaču.
